Linera Client Setup Instructions

Required Import Map

According to Linera documentation, you need an import map for the WASM client to work.

Add this to your frontend/src/app/layout.tsx:

<script type="importmap">
{
  "imports": {
    "@linera/client": "https://cdn.example.com/@linera/client@0.15.3/index.js"
  }
}
</script>

Official Linera Client Documentation

  • Frontend Interactivity Guide
  • Linera Client GitHub

Integration Steps

  1. Install: npm install @linera/client@0.15.3
  2. Add import map to HTML
  3. Import and initialize in frontend component
  4. Call client.frontend().application(APP_ID) to get backend
  5. Use backend.query() for GraphQL queries

GraphQL API

Visit http://localhost:8080 when running linera service to access the GraphQL IDE.
